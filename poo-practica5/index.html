Actividad
Gestión de Cuentas Bancarias: Crea una clase CuentaBancaria que tenga propiedades para nombre del titular, saldo y numero de cuenta. Utiliza métodos set para establecer el nombre del titular y el saldo, y métodos get para obtener estos valores. Implementa un método para permitir la realización de depósitos en la cuenta y otro método para permitir la realización de retiros. Asegúrate de que el método para realizar retiros no permita retirar más dinero del disponible en la cuenta.
A realizar:
  • Desarrollar los respectivos metodos Getters y Setters.
