Migraciones
Las migraciones son una forma de actualizar la estructura de la base de datos de una aplicación de .NET Core de manera incremental y controlada.
Cuando trabajas con Entity Framework Core, defines tus modelos de datos como clases de C# que representan las entidades en tu base de datos. Las migraciones permiten traducir los cambios que haces en estos modelos de datos en cambios en la estructura de la base de datos subyacente, como crear, modificar o eliminar tablas, columnas, índices, etc.

Practica
  • Crear un proyecto de MVC.
  • Modificar nuestro archivo appsettings.json
  • Instalacion del paquete NuGet Microsoft.EntityFrameworkCore.SqlServer
Este paquete proporciona los componentes necesarios para que Entity Framework Core pueda conectarse, interactuar y manipular datos en una base de datos SQL Server.
  • Instalacion del paquete NuGet Microsoft.EntityFrameworkCore.Tools
Este paquete proporciona herramientas de línea de comandos (CLI) para trabajar con Entity Framework Core (EF Core) dentro de proyectos de C#. Estas herramientas son útiles durante el desarrollo de una aplicación para realizar tareas como la creación de migraciones de base de datos, aplicar esas migraciones, generar código de modelos a partir de la base de datos, entre otras cosas.

  • Crearemos una clase MDepartamento dentro de la carpeta de nuestros modelos.
  • Crearemos una clase MEmpleado dentro de la carpeta de nuestros modelos.
  • Crearemos una carpeta Data la cual contendra una clase AppDbContext.

Actividad
En este proyecto, nos embarcaremos en la emocionante tarea de desarrollar una página web dedicada a la venta de videojuegos. Nuestro objetivo principal es crear una plataforma intuitiva y atractiva que permita a los usuarios explorar y adquirir una amplia variedad de títulos de videojuegos.

Cada videojuego estará categorizado según su categoria, lo que facilitará la navegación y búsqueda para nuestros usuarios. Además, implementaremos una función importante: la identificación de las consolas compatibles con cada juego. Esto garantizará que los usuarios puedan encontrar rápidamente los videojuegos que sean compatibles con sus consolas.

A lo largo de este proyecto, trabajaremos en el diseño de la interfaz de usuario, la implementación de sistemas de navegación eficientes, la integración de una base de datos para gestionar las características únicas de cada título. Además, nos aseguraremos de que nuestra página web sea fácil de usar y esté optimizada para una experiencia de usuario fluida.

Este proyecto nos brindará la oportunidad de aplicar nuestros conocimientos en desarrollo web, diseño de interfaz de usuario y gestión de bases de datos en un contexto práctico y emocionante. ¡Esperamos crear una plataforma que inspire a los jugadores y haga que la experiencia de compra de videojuegos sea aún más emocionante y accesible para todos!
