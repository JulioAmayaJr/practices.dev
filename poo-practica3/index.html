Desarrollar los ejercicios del laboratorio.

Ejercicio El Sistema de Gestión de Tienda de Mascotas es una aplicación diseñada para administrar el inventario y las ventas de una tienda de mascotas. Este sistema permite a los propietarios de la tienda llevar un registro detallado de las mascotas disponibles para la venta, así como gestionar las transacciones de compra realizadas por los clientes.

A realizar:
  • Crear Clase Mascota con Getters y Setters.
  • Crear Clase TiendaMascota para contener una lista de mascotas disponibles con los metodos para agregar, eliminar y buscar mascotas.
  • Programar la logica del metodo agregar.
  • Programar la logica para eliminar una mascota.
  • Programar el metodo para la busqueda de una mascota por medio de su id.
  • Crear clase cliente con sus respectivos Getters y Setters.
  • Agregar un metodo a la clase TiendaMascota para que un cliente pueda comprar una mascota.
  • Crear un metodo para poder ver las mascotas vendidas
