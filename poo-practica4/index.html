Herencia
En programación orientada a objetos, la herencia es un mecanismo utilizado para alcanzar algunos de los objetivos más preciados en el desarrollo de software como lo son la reutilización y la extensibilidad.

Ejercicio En una tienda de electrónica, se necesita implementar un sistema para gestionar las órdenes de los clientes. La tienda ofrece una variedad de productos, como televisores, laptops, smartphones y accesorios.

La empresa solicita lo siguiente:

Gestión de productos: Se deben listar los productos disponibles, incluyendo su nombre, precio y cantidad en stock.

Gestión de pedidos del cliente: Se debe permitir a los clientes realizar pedidos de productos. Cada pedido debe contener el producto solicitado, la cantidad y el cliente que realizó el pedido.

Gestión de inventario: Se debe mantener actualizado el inventario de productos después de cada pedido realizado.

El proceso que maneja la empresa es el siguiente:
  • El cliente selecciona los productos que desea comprar, indicando la cantidad de cada uno.
  • El sistema registra el pedido, verifica la disponibilidad de los productos en el inventario y actualiza el inventario después de cada pedido.
  • El cliente recibe una confirmación del pedido junto con el total a pagar.
